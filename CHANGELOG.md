### Added

- Admin settings integration with a dedicated Teams Governance page.
- Admin governance API endpoint for paginated team listing.
- Team search by team name, creator display name, and creator username.
- Per-team details endpoint with lazy-loaded connected resources.
- Team details panel in the UI with provider counts and resource links.
- Team share (Files) yes/no indicator in team details.
- Team row identifier and details capability flags in API responses.
- Unit test coverage for API search forwarding and details endpoint forwarding.
